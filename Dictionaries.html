Dictionaries

Overview

A dictionary is a collection of key-value pairs where each key maps to a specific value. It provides fast lookup based on keys, similar to arrays, but it allows for dynamic resizing like linked lists.

Methods of a Dictionary

Common methods used with dictionaries include:

  • .Add() - Adds the specified key and value pair to the dictionary.
  • .TryGetValue() - Gets the value associated with the specified key.
  • .ContainsKey() - Determines whether the dictionary contains the specified key.
  • .ContainsValue() - Determines whether the dictionary contains a specific value.
  • .Remove() - Removes the value with the specified key from the dictionary.

Example Usage (C#)

                    
using System;
using System.Collections.Generic;

class Program {
    static void Main(string[] args) {
        Dictionary ageDict = new Dictionary();
        ageDict.Add("John", 30);
        ageDict.Add("Alice", 25);
        ageDict.Add("Bob", 35);

        Console.WriteLine("Age of John: " + ageDict["John"]);
        Console.WriteLine("Does the dictionary contain Alice? " + ageDict.ContainsKey("Alice"));
    }
}
