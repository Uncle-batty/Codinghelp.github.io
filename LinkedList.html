Linked-List

Overview

In computer science, a linked list is a linear data structure consisting of a sequence of elements, called nodes. Each node contains a data element and a reference (or link) to the next node in the sequence.

Singly Linked Lists: Each node contains a data element and a single reference to the next node in the sequence. The last node points to a null reference.

Doubly Linked Lists: Each node contains references to both the next and previous nodes, allowing bidirectional traversal.

Circular Linked Lists: The last node points back to the first node, forming a circular structure.

Common Methods

  • .AddFirst(Value) - Adds a value as the first node.
  • .AddLast(Value) - Adds a value as the last node.
  • .AddAfter(index, Value) - Adds a value after a specified index.
  • .Remove(Value) - Removes a node with the specified value.
  • .Contains(Value) - Checks if the list contains a specific value.
  • .Find(Value) - Finds the first occurrence of a value.
  • .First - Gets the first node.
  • .Last - Gets the last node.

Implementing a Linked List

Here's how you can code a simple linked list from scratch in C#:


using System;

public class Node {
    public int Value { get; set; }
    public Node Next { get; internal set; }

    public Node(int value) {
        Value = value;
        Next = null;
    }
}

public class MyLinkedList {
    public Node Head;

    public MyLinkedList() {
        Head = null;
    }

    // Methods to manipulate the list (Add, Remove) go here...

    public void Reverse() {
        Node previous = null, current = Head, next = null;
        while (current != null) {
            next = current.Next;
            current.Next = previous;
            previous = current;
            current = next;
        }
        Head = previous;
    }
}
