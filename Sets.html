Hash Sets (aka Sets)

Overview

A hash set is a data structure used to store unique elements efficiently. It utilizes a hashing function to map elements to indexes in an array-like structure.

Methods of a Hash Set

Common methods used with hash sets include:

  • .Add() - Adds the specified element to the hash set.
  • .Contains() - Checks whether the hash set contains a specific element.
  • .Remove() - Removes the specified element from the hash set.

Example Usage (C#)

                    
using System;
using System.Collections.Generic;

class Program {
    static void Main(string[] args) {
        HashSet numbers = new HashSet();
        numbers.Add(1);
        numbers.Add(2);
        numbers.Add(3);
        numbers.Add(3); // Adding duplicate, but it won't be stored

        Console.WriteLine("HashSet contains 2? " + numbers.Contains(2)); // Output: True
        Console.WriteLine("HashSet contains 4? " + numbers.Contains(4)); // Output: False
    }
}

Implementing a Hash Set

Here's how you can implement a hash set from scratch in C#:


using System;

public class MyHashSet {
    private bool[] data = new bool[1000001]; // Assuming a maximum of 1,000,001 elements
    const int size = 1000001;

    public MyHashSet() {
    }

    // Hashing function to get index
    public int GetIndex(string key) {
        int hash = 0;
        foreach (char c in key) {
            hash = (hash * 31 + c) % size;
        }
        return hash;
    }

    // Add method
    public void Add(string key) {
        int index = GetIndex(key);
        data[index] = true;
    }

    // Remove method
    public void Remove(string key) {
        int index = GetIndex(key);
        data[index] = false;
    }

    // Contains method
    public bool Contains(string key) {
        int index = GetIndex(key);
        return data[index];
    }
}

class Program {
    static void Main() {
        MyHashSet set = new MyHashSet();
        set.Add("apple");
        set.Add("banana");
        Console.WriteLine("HashSet contains 'apple'? " + set.Contains("apple")); // Output: True
        set.Remove("apple");
        Console.WriteLine("HashSet contains 'apple' after removal? " + set.Contains("apple")); // Output: False
    }
}

Problem and Solution

Problem: Given two arrays, write a function to compute their intersection.

Solution: One approach is to use hash sets to store elements from one array, then iterate through the other array and check for intersection. Here's a sample solution in C#: 


using System;
using System.Collections.Generic;

class Program {
    static int[] Intersection(int[] nums1, int[] nums2) {
        HashSet set = new HashSet();
        foreach (int num in nums1) {
            set.Add(num);
        }

        List intersection = new List();
        foreach (int num in nums2) {
            if (set.Contains(num)) {
                intersection.Add(num);
                set.Remove(num); // Remove duplicates
            }
        }

        return intersection.ToArray();
    }

    static void Main() {
        int[] nums1 = { 1, 2, 2, 1 };
        int[] nums2 = { 2, 2 };
        int[] result = Intersection(nums1, nums2);
        Console.WriteLine("Intersection of nums1 and nums2: " + string.Join(", ", result)); // Output: 2
    }
}
