Lists/Dynamic Array

Overview: Lists are dynamic data structures that grow and shrink, providing flexible and efficient ways to manage ordered data. Dynamic Arrays enhance this by adjusting the storage space as items are added or removed.

Methods of a List: Lists provide various methods to manipulate collections efficiently. The most commonly used methods include .Add(), .Remove(), and .Find().

Example of a Dynamic Array: Below is an example of how to implement a Dynamic Array in C#. 


    using System;
    using System.Collections.Generic;

    class DynamicArray {
        private T[] items;
        private int count;
        
        public DynamicArray(int initialCapacity) {
            items = new T[initialCapacity];
            count = 0;
        }

        public void Add(T item) {
            if (count == items.Length) Resize();
            items[count++] = item;
        }

        private void Resize() {
            T[] newItems = new T[items.Length * 2];
            for (int i = 0; i < items.Length; i++) {
                newItems[i] = items[i];
            }
            items = newItems;
        }

        // Other methods omitted for brevity...
    }

Using a List: The following example demonstrates creating and manipulating a list in C#. 


    using System;
    using System.Collections.Generic;

    class Program {
        static void Main() {
            List myList = new List();
            myList.Add("Apple");
            myList.Add("Banana");
            myList.Add("Cherry");

            foreach (var item in myList) {
                Console.WriteLine(item);
            }
        }
    }
