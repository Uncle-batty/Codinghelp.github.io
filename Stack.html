Stacks

Overview

A stack is a linear data structure that follows the Last In, First Out (LIFO) principle. It has two primary operations: push, which adds an element to the top of the stack, and pop, which removes the top element from the stack.

Methods of a Stack

Common methods used with stacks include:

  • .Push() - Adds an element to the top of the stack.
  • .Pop() - Removes and returns the top element from the stack.
  • .Peek() - Returns the top element from the stack without removing it.
  • .IsEmpty() - Checks whether the stack is empty.

Example Usage (C#)

                    
using System;
using System.Collections.Generic;

class Program {
    static void Main(string[] args) {
        Stack stack = new Stack();
        stack.Push(1);
        stack.Push(2);
        stack.Push(3);

        Console.WriteLine("Top element of stack: " + stack.Peek()); // Output: 3
        Console.WriteLine("Popping element from stack: " + stack.Pop()); // Output: 3
    }
}
