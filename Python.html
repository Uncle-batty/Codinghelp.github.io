Python

Python is a high-level language that is very easy to learn and is often the first language new programmers start with.

Hello World – Output

Python uses the print() function to output to the console.

print("HELLO WORLD!")

Output: HELLO WORLD!

Variables & User Input

Variables in Python are easy to declare because Python infers the type automatically.

isSwitchOn = True       # Boolean
firstInitial = 'L'      # Character
surname = "Batty"       # String
value = 23              # Integer
cash = 120.55           # Float

User input is taken using the input() function. All input returns a string, so convert types when needed.

name = input("Enter your name: ")
age = int(input("Enter your age: "))

print("Your name is " + name + " you are " + str(age) + " years old")

Output:
Enter your name: Louis
Enter your age: 20
Your name is Louis you are 20 years old

Operators

Python supports common arithmetic operators:

+   # Addition or string concatenation
-   # Subtraction
*   # Multiplication
/   # Division
%   # Modulus (remainder)
//  # Floor division
**  # Exponent

Examples:

2 + 2      # 4
10 - 20    # -10
3 * 10     # 30
25 / 5     # 5.0
10 % 3     # 1
11 // 2    # 5
2 ** 3     # 8

Conditionals

if condition:
    # Code
elif other_condition:
    # Code
else:
    # Code

Example:

age = 18

if age >= 18:
    print("Adult")
else:
    print("Minor")

Loops

For Loop:

for i in range(0, 5):
    print(i)

While Loop:

count = 0
while count < 5:
    print(count)
    count += 1

Classes & Object-Oriented Programming

Defining Classes

class Car:
    def __init__(self, make, model):
        self.make = make
        self.model = model

    def start(self):
        print("Car started.")

Creating Objects

myCar = Car("Toyota", "Camry")
myCar.start()

Inheritance

class ElectricCar(Car):
    def __init__(self, make, model, battery):
        super().__init__(make, model)
        self.battery = battery

Polymorphism

class Animal:
    def speak(self):
        print("Animal sound")

class Dog(Animal):
    def speak(self):
        print("Bark")

class Cat(Animal):
    def speak(self):
        print("Meow")

File Handling

# Write to file
with open("file.txt", "w") as file:
    file.write("Hello World")

# Read from file
with open("file.txt", "r") as file:
    print(file.read())

Modules

import math

print(math.sqrt(25))
