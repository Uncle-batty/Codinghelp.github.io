Arrays

Overview

Arrays in computer science store elements of the same type in contiguous memory locations, allowing efficient index-based access. Unlike linked lists, arrays have a fixed size that must be specified at creation and cannot be changed.

Methods of an array

Common methods used with arrays include:

  • .Length - Returns the number of elements in the array.
  • .GetValue(index) - Retrieves the value at the specified index.
  • .SetValue(value, index) - Sets the value at the specified index.
  • .CopyTo(array, index) - Copies elements to another array starting at the specified index.
  • .IndexOf(value) - Returns the index of the first occurrence of the specified value.
  • .Reverse() - Reverses the order of elements in the array.
  • .Sort() - Sorts elements in ascending order.
  • .Clear() - Removes all elements from the array.

Example Usage (C#)

                    
using System;

class Program {
    static void Main(string[] args) {
        int[] numbers = { 1, 2, 3, 4, 5 };
        Console.WriteLine("First element: " + numbers[0]);
        Console.WriteLine("Last element: " + numbers[numbers.Length - 1]);
    }
}
