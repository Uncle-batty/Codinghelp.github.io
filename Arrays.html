Arrays

Overview

Arrays: In computer science, an array is a data structure that stores a collection of elements of the same type. Unlike linked lists, which use references to connect nodes, arrays store elements in contiguous memory locations. This allows for direct access to elements using index notation.

Arrays have a fixed size determined at the time of declaration, and the size cannot be changed dynamically during program execution. This means that arrays cannot grow or shrink in size once created.

Despite their fixed size limitation, arrays offer several advantages, including constant-time access to elements, which makes them suitable for scenarios where frequent random access is required. They are also memory-efficient, as they allocate contiguous memory blocks for elements.

However, arrays have limitations when it comes to dynamic resizing and insertion/deletion operations. When elements need to be added or removed frequently, arrays may not be the most efficient choice, as these operations often require shifting elements and reallocating memory.

Overall, arrays are a fundamental data structure in computer science, widely used in various applications and algorithms, including sorting algorithms, matrix operations, and representing collections of data.

Methods of an array

These are common methods used with arrays:

  • .Length: Gets the length of the array, which is the total number of elements it can hold.
  • .GetValue(index): Retrieves the value at the specified index in the array.
  • .SetValue(value, index): Sets the value at the specified index in the array.
  • .CopyTo(array, index): Copies the entire array to another array starting at the specified index.
  • .IndexOf(value): Searches for the specified value in the array and returns the index of its first occurrence.
  • .Reverse(): Reverses the order of the elements in the array.
  • .Sort(): Sorts the elements in the array in ascending order.
  • .Clear(): Removes all elements from the array.

How to use Arrays (C#)

You can use arrays in C# to store and manipulate collections of elements. Here's a basic example: 

                
        using System;
    
        class Program
        {
            static void Main(string[] args)
            {
                // Declare and initialize an array of integers
                int[] numbers = { 1, 2, 3, 4, 5 };
    
                // Access elements using index notation
                Console.WriteLine("First element: " + numbers[0]);
                Console.WriteLine("Second element: " + numbers[1]);
                Console.WriteLine("Last element: " + numbers[numbers.Length - 1]);
            }
        }
