Strings

Overview

Strings in programming are sequences of characters used to store and manipulate text. They can include letters, numbers, and symbols. In many languages, strings are immutable, meaning once they are created, their contents cannot be changed.

Common String Operations

Some frequently used operations on strings include:

  • .Length - Returns the length of the string.
  • .Substring(startIndex, length) - Returns a new string that is a substring of the original string, starting from the specified index and extending a specified number of characters.
  • .IndexOf(substring) - Returns the index within the string of the first occurrence of the specified substring.
  • .Replace(oldValue, newValue) - Returns a new string in which all occurrences of a specified string in the current instance are replaced with another specified string.
  • .ToLower() - Returns a copy of the string converted to lowercase.
  • .ToUpper() - Returns a copy of the string converted to uppercase.
  • .Trim() - Removes all leading and trailing white-space characters from the current string.
  • .Split(separator) - Splits the string into substrings based on the characters in the specified separator and returns an array of these substrings.

Example Usage (C#)

                    
using System;

class Program {
    static void Main(string[] args) {
        string greeting = "Hello, World!";
        Console.WriteLine("Original string: " + greeting);
        Console.WriteLine("Lower case: " + greeting.ToLower());
        Console.WriteLine("Upper case: " + greeting.ToUpper());
    }
}
